# Looping Statements in C

Loops allow repetitive execution of a block of code until a specified condition is met.

**Types of loops:**
- **`for` loop** — Best when the number of iterations is known in advance
- **`while` loop** — Best when repetition depends on a condition being true
- **`do-while` loop** — Similar to while, but guarantees at least one execution
- **Nested loops** — Loops within loops for multi-dimensional processing and patterns
